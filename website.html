Description of the github workflow

The git hub workflow allows teams to maintain, collaborate & edit code using git reposotories

What is a git branch?

Git branch is a method used to track the code of the application you're working on while tracking it's versions

what is a pull request?

A pull request is a method used to discuss changes made to the code base before integration with main branch/project

Description of the Unix shell

The unix shell allows for easier management of files & directories without a GUI - as well as providing the ability to automate tasks

How to list files in a directory

By usig the "ls" phrase

How to show and change the working directory

You can show and change the working directory by e.g. typing "cd Desktop", once you select enter to change to the Desktop. Then type "ls" to list the files/folders in the directory

How to create a new directory

You can create a new directory using the keyword "mkdir"

How to create new files

You can create a new file using the keyword "nano"

How to remove files

You can remove files using the key word "rm"
